<h1 align="center">🎵 ArduinoThing 🎵</h1>

Have you ever wanted to make your own custom DIY Car Thing? Well, it's great that you've come to the right place. This is ArduinoThing, a small program written in MicroPython and C++.

This project is compatible with both ESP32 (WROOM32) and Arduino UNO, and you can adjust whatever you want to display on an OLED or many other devices.
